Extracting Amazon product Data

It is a tool to extract data about a specific product from the Amazon product page The extracted data will be

  • Save data as .xlsx
    • Basic information
      • Product name
      • Product cost
      • About items
      • Number of global ratings
    • Rating information
    • Screenshot
      • Save screenshot of product
    • Comments/customer reviews
      • Data
        • Name
        • Rating
        • Review title
        • Date
        • Comment
      • User can specify specific time (in seconds) to extract comments/reviews, this will save time, otherwise scrapers will extract all comments, which can take a long time due to some products on amazon have 1k+ comments And the scraper needs to go over 100+ pages because amazon shows 10 comments/page
